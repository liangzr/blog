Option 1: 仅实现登陆

最简单的实现手动的登陆，可以用expect工具来实现，在Ubuntu下需要下载：
sudo apt-get install expect
Expect中最关键的四个命令是：

  • send：用于向进程发送字符串
  • xpect：从进程接收字符串
  • spawn：启动新的进程
  • interact：允许用户交互
